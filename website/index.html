[ What it includes ]

infra/

Compose profiles, reverse proxy, database bootstrap, and runtime defaults.

openclaw/

Agent and cron config templates with deterministic-first execution patterns.

automation/

Dispatcher/watcher modules and config-driven orchestration scaffolds.

[ Architecture ]

      |

  Postgres + Redis

[ Quickstart ]

git clone https://github.com/chhotu-claw/ai-ops-starter-kit.git
cd ai-ops-starter-kit
make bootstrap
make status

Tip: use SKIP_COMPOSE=1 make bootstrap for seed-only configuration.
